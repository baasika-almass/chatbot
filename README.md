# 🤖 PyBot — Simple Chatbot

A rule-based chatbot built in Python using regex pattern matching.
This is a beginner introduction to NLP (Natural Language Processing) concepts.

## Features
- Greets you by name
- Tells the current time and date
- Tells random jokes
- Solves simple math expressions (e.g. "5 + 3")
- Responds to greetings, farewells, and thanks
- Fallback response for unrecognized input

## How to Run
```bash
python chatbot.py
```

## Example
🤖 PYBOT — Simple Chatbot
What's your name? Baasika
PyBot: Hi Baasika! Ask me your name, the time, a joke,

or give me a math problem like '5 + 3'.

Type 'bye' to exit.
Baasika: what is your name

PyBot: I'm PyBot — a simple rule-based chatbot built in Python!
Baasika: 12 + 8

PyBot: That equals 20
Baasika: tell me a joke

PyBot: Why do programmers prefer dark mode? Because light attracts bugs!
Baasika: bye

PyBot: Goodbye Baasika! Have a great day! 👋
